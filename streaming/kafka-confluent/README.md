###  instructions
[X] download self-managed zip file from https://www.confluent.io/download/plans#confluent-platform
[X] Save config.sh, start.sh, stop.sh, & tail-logs.sh at confluent folder. Example: C:/confluent/start.sh, C:/confluent/bin/
[X] Update JAVA_HOME & CONFLUENT_HOME in config.sh


### Start Container
`````
./start.sh
`````

### Stop Containers
`````
./stop.sh
`````

### Tail logs
`````
./tail-logs.sh
`````
